THE VAULT OF ECHOES - LOCATIONS TO ADD

HOW TO USE THIS FILE

1. Mark each desired location on your map screenshot with a number.
2. Fill in the matching numbered section below.
3. For Image, write the image filename or attach the image when you send this file.
   Location images are stored in public/images/locations.
4. Population and Link are optional. Leave either line blank when it is not needed.
5. Link may point to Steam, YouTube, a website, or another external page.
6. Copy and paste another blank section if you need more locations.
